I.12 DisguiseLogger

A logger which masks passwords in URI strings - i.e: [echo] hide password from URI http://foo:*****@example.com/baz?one=two

To use the DisguiseLogger, start Phing with this command: phing -logger "Phing\\Listener\\DisguiseLogger"
